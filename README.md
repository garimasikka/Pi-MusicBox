# π Music Box

A minimalist web application that transforms the digits of π (pi) into a visual and auditory experience. Each digit of pi is associated with a specific musical note and color, creating a unique synesthetic experience as the infinite sequence plays.

## Overview
Pi Music Player converts each digit of π into musical notes while displaying the digits in a corresponding color spectrum against a minimalist black background. The application is designed to be responsive, working seamlessly on both mobile devices and larger screens.

## Features
- **Visual representation**: Each digit appears in a unique color as it plays
- **Musical interpretation**: Digits are mapped to musical notes using a scale system
- **Minimalist design**: Clean black background with focus on the digits and sound
- **Play/Pause functionality**: Simple control to start or stop the sequence
- **Long-playing capability**: Contains enough digits to play for an extended period without repeating

## How It Works
1. The application maps each digit of π (0-9) to a specific note in a musical scale
2. As digits are played sequentially, they briefly appear on screen in their assigned color
3. The corresponding musical note plays simultaneously with the visual appearance
4. The sequence continues until paused

## Technical Implementation
- Pure JavaScript for audio generation using the Web Audio API
- CSS animations for smooth visual transitions
- Responsive design for cross-device compatibility
- No external dependencies required

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/pi-music-player.git
   ```

2. Navigate to the project directory:
   ```
   cd pi-music-player
   ```

3. Open `index.html` in your web browser to run the application locally
